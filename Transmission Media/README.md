# Transmission Media
# 📖 Overview

**Transmission Media** is the communication path through which data travels from one network device to another.

Whenever two devices communicate over a network, the transmitted data must pass through a medium. This medium can either be a **physical cable** or a **wireless communication channel**.

Understanding transmission media is fundamental because it determines:

- Communication speed
- Transmission distance
- Reliability
- Security
- Cost
- Overall network performance

This module focuses on the **fundamentals of Transmission Media** and introduces the different types of guided transmission media used in modern computer networks.
