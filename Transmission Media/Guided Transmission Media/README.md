# Guided Transmission Media
# 📖 Overview

**Guided Transmission Media** refers to a type of transmission medium in which data travels through a **physical path**, such as cables or wires.

Unlike wireless communication, guided media confines signals within a dedicated medium, resulting in reliable communication, reduced interference, and improved security.

Guided transmission media forms the foundation of most modern **Local Area Networks (LANs)**, enterprise networks, and data center infrastructures.

# 🔍 Characteristics of Guided Transmission Media

Guided transmission media has the following characteristics:

- Uses physical cables to transmit data
- Provides a dedicated communication path
- Supports high-speed communication
- Less susceptible to electromagnetic interference (depending on cable type)
- More secure than wireless communication
- Suitable for both short and long-distance communication

---

# 📊 Comparison of Guided Transmission Media

| Feature | Twisted Pair | Coaxial Cable | Fiber Optic Cable |
|----------|--------------|---------------|-------------------|
| Transmission Signal | Electrical | Electrical | Light |
| Speed | Medium to High | High | Very High |
| Distance | Short | Medium | Long |
| EMI Resistance | Moderate (UTP), High (STP) | High | Excellent |
| Cost | Low | Medium | High |
| Common Uses | Ethernet LANs | Cable TV, Broadband | ISPs, Data Centers, Backbone Networks |

# 💡 Why Guided Transmission Media?

Guided media is preferred because it offers:

- Reliable communication
- High data transmission speed
- Stable network performance
- Better security
- Reduced signal interference
- Predictable communication quality
