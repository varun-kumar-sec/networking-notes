# Guided Transmission Media
# 📖 Overview

**Guided Transmission Media** refers to communication channels in which data signals travel through a **physical medium**, such as cables or wires.

Unlike wireless communication, guided media provides a dedicated path for data transmission, resulting in higher reliability, better security, and lower interference.

Guided transmission media forms the backbone of most modern Local Area Networks (LANs), enterprise networks, and data centers.

# 🔌 What is Guided Transmission Media?

**Guided Transmission Media** is a type of transmission medium in which data travels through a **physical path** made of cables or wires.

The physical medium guides the signal from the sender to the receiver, which is why it is called **Guided Media**.

Depending on the cable used, data may travel as:

- Electrical Signals
- Light Signals

Unlike wireless communication, the signal remains confined within the transmission medium.

---

# ❓ Why is Guided Media Required?

Guided Media is widely used because it provides:

- Reliable communication
- High-speed data transmission
- Low signal interference
- Better security
- Stable network performance

It is the preferred transmission medium for environments where speed, reliability, and security are important.

---

# ⚙️ How Does Guided Media Work?

The communication process is as follows:

1. A sender generates data.
2. The data is converted into electrical or light signals.
3. The signals travel through a physical cable.
4. The receiver receives the signals.
5. The signals are converted back into usable data.

Example:

```
Computer A
     │
     ▼
Guided Transmission Media
     │
     ▼
Computer B
```

Since the signal remains inside the cable, it experiences less interference compared to wireless communication.

# 📌 Characteristics of Guided Transmission Media

Guided media has the following characteristics:

- Uses physical cables or wires
- Provides a dedicated communication path
- Offers high reliability
- Less affected by external interference
- Supports high-speed communication
- Suitable for both short and long-distance communication (depending on the cable type)

---

# 📂 Types of Guided Transmission Media

Guided Transmission Media is mainly divided into three types:

## 1. Twisted Pair Cable

- Most commonly used networking cable
- Used in Ethernet LANs
- Inexpensive and easy to install

---

## 2. Coaxial Cable

- Better shielding than twisted pair
- Used in cable television and broadband communication
- Supports higher bandwidth than twisted pair

---

## 3. Fiber Optic Cable

- Uses light instead of electrical signals
- Extremely high speed
- Long-distance communication
- Highly resistant to electromagnetic interference

Each of these cable types will be covered in detail in the following sections.

# ❌ Disadvantages

- Cable damage may interrupt communication
