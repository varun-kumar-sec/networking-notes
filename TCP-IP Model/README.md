# TCP/IP Model
# 📖 Overview

The **TCP/IP (Transmission Control Protocol/Internet Protocol) Model** is the **standard networking model used by the Internet** and modern computer networks.

It provides a practical framework for how data is transmitted, addressed, routed, and received across interconnected networks.

Unlike the OSI Model, which is primarily a conceptual reference model, the TCP/IP Model is **implemented in real-world networking devices and operating systems**. It defines a set of protocols that enable communication between computers regardless of their hardware or operating system.

Today, virtually every device connected to the Internet uses the TCP/IP protocol suite.

# 🏗️ The Four Layers

The TCP/IP Model consists of four layers:

| Layer | Primary Responsibility |
|------:|------------------------|
| 4 | Application Layer |
| 3 | Transport Layer |
| 2 | Internet Layer |
| 1 | Network Access Layer |

Each layer performs a specific networking function and works together to provide reliable communication across networks.
