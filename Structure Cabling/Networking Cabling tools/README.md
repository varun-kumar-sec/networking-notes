# Network Cabling Tools
# 📖 Overview

**Network Cabling Tools** are specialized hand tools used to install, terminate, repair, and maintain network cables.

These tools ensure that Ethernet cables are prepared correctly before connectors are attached, resulting in reliable network communication and long-lasting cable performance.

In this module, we will learn about the two most commonly used tools in Ethernet cable installation:

- **Cable Stripper**
- **Crimping Tool**

These tools are essential for creating custom Ethernet cables used in Local Area Networks (LANs).

# 📚 Topics Covered

| Tool | Description |
|------|-------------|
| Cable Stripper | Removes the outer insulation of Ethernet cables without damaging the internal conductors |
| Crimping Tool | Attaches RJ45 or RJ11 connectors to Ethernet and telephone cables |

---

# 🌍 Why are Network Cabling Tools Important?

Proper networking tools help technicians:

- Prepare Ethernet cables correctly
- Prevent damage to internal conductors
- Create reliable cable terminations
- Reduce installation errors
- Improve network reliability
- Speed up cable installation and maintenance

Using the correct tool ensures better network performance and longer cable life.
