# Uses of Straight-through Cable
# 📖 Overview

A **Straight-through Cable** is the most commonly used Ethernet cable in modern computer networks.

It is used to connect **different types of networking devices**, allowing them to communicate efficiently.

A straight-through cable is created when **both ends of the Ethernet cable use the same wiring standard**, either **T568A–T568A** or **T568B–T568B**.

Today, straight-through cables are widely used in homes, offices, schools, data centers, and enterprise networks.

# 🔗 What is a Straight-through Cable?

A **Straight-through Cable** is an Ethernet cable where **both ends are terminated using the same wiring standard**.

Examples:

```
T568A → T568A
```

or

```
T568B → T568B
```

Since both ends use the same wiring sequence, the transmit (TX) and receive (RX) wire pairs remain unchanged.

---

# ❓ Why is a Straight-through Cable Used?

A Straight-through Cable is used to connect **different types of networking devices**.

It provides a reliable connection for Ethernet communication and is the standard cable used in most Local Area Networks (LANs).

---

# 🌍 Common Uses of Straight-through Cables

Straight-through cables are commonly used for the following connections:

## Computer ↔ Switch

The most common Ethernet connection in LANs.

---

## Computer ↔ Hub

Used in older Ethernet networks where hubs are still deployed.

---

## Switch ↔ Router

Connects a network switch to a router for Internet or inter-network communication.

---

## Router ↔ Modem

Connects a router to a modem for Internet access.

---

## Access Point ↔ Switch

Connects a wireless access point to the wired network.

---

## Printer ↔ Switch

Allows network printers to communicate with multiple computers.
