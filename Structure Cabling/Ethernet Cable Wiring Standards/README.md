# Ethernet Cable Wiring Standards
# 📖 Overview

**Ethernet Cable Wiring Standards** define the correct arrangement of wires inside an Ethernet cable before they are terminated using an **RJ45 connector**.

Following standardized wiring ensures compatibility, reliable communication, and proper data transmission between networking devices.

The two internationally accepted wiring standards are:

- **T568A**
- **T568B**

These standards are also used to create the two most common Ethernet cable types:

- **Straight-through Cable**
- **Crossover Cable**

Understanding these standards is essential for network installation, troubleshooting, and Ethernet cable creation.

# 🌈 Ethernet Wiring Standards

Ethernet cables contain **eight copper conductors** arranged in a specific order.

Following the correct wire sequence ensures:

- Reliable communication
- Proper signal transmission
- Compatibility with networking devices
- Compliance with international networking standards

The two standard wiring schemes are:

- **T568A**
- **T568B**

---

# 🔀 Cable Types

## Straight-through Cable

A Straight-through Cable is created when **both ends use the same wiring standard**.

Examples:

- T568A ↔ T568A
- T568B ↔ T568B

Used to connect **different networking devices**.

Examples:

- Computer ↔ Switch
- Switch ↔ Router
- Router ↔ Modem

---

## Crossover Cable

A Crossover Cable is created when **one end uses T568A** and **the other end uses T568B**.

Example:

- T568A ↔ T568B

Used to connect **similar networking devices**.

Examples:

- Computer ↔ Computer
- Switch ↔ Switch
- Router ↔ Router

> **Note:** Modern networking devices often support **Auto MDI-X**, which automatically detects the cable type, reducing the need for crossover cables.
