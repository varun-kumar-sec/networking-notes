# T568B Wiring Standard (Commonly Used for Straight-through Cables)
# 📖 Overview

**T568B** is one of the two standard wiring schemes defined by the **TIA/EIA-568** standard for terminating Ethernet cables.

It specifies the order of the eight conductors inside a twisted pair cable before they are inserted into an **RJ45 connector**.

When **both ends** of an Ethernet cable are terminated using **T568B**, the cable becomes a **Straight-through Cable**.

T568B is the most commonly used wiring standard in modern Ethernet networks.

# 🔧 What is T568B?

**T568B** is a standardized wiring arrangement used for terminating Ethernet cables with **RJ45 connectors**.

It defines the correct sequence of the eight conductors inside a twisted pair cable.

T568B is widely used because it has become the preferred wiring standard for most commercial Ethernet installations.

# 🔗 What is a Straight-through Cable?

A **Straight-through Cable** is created when **both ends** of an Ethernet cable use the **same wiring standard**.

Examples:

```
End A → T568A
End B → T568A
```

or

```
End A → T568B
End B → T568B
```

Since both ends use the same wire sequence, the transmit and receive pairs remain unchanged.

---

# 🌍 Applications

Straight-through cables are commonly used to connect **different types of networking devices**, such as:

- Computer ↔ Switch
- Computer ↔ Hub
- Switch ↔ Router
- Router ↔ Modem
- Access Point ↔ Switch

These are the most commonly used Ethernet cables in homes, offices, and enterprise networks.
