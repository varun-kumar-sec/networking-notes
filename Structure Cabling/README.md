# Structured Cabling
# 📖 Overview

**Structured Cabling** is a standardized approach to designing, installing, and managing network cabling infrastructure. Instead of connecting devices using random or unorganized cables, structured cabling follows internationally accepted standards to create a reliable, scalable, and easy-to-maintain network.

It serves as the physical foundation of modern computer networks and supports various communication services, including Ethernet, Voice over IP (VoIP), Internet connectivity, CCTV, and telecommunication systems.

This module introduces the fundamentals of structured cabling, the tools used for cable installation, Ethernet wiring standards, and the process of creating Ethernet cables.
