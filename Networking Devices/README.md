# Networking Devices
# 📖 Overview

Networking devices are specialized hardware components that enable communication between computers, servers, printers, mobile devices, and other network-enabled systems.

Each networking device performs a specific role, such as connecting devices, forwarding data, amplifying signals, routing packets, or enabling Internet connectivity.

Understanding these devices is fundamental before learning advanced networking topics such as:

- OSI Model
- TCP/IP Model
- IP Addressing
- Subnetting
- Routing
- Switching
- Network Security

This module introduces the most common networking devices used in modern computer networks.

# 📑 Topics Covered

This module includes the following networking devices:

| Device | Description |
|----------|-------------|
| **NIC (Network Interface Card)** | Connects a computer to a network. |
| **Repeater** | Regenerates weak signals to extend network distance. |
| **Hub** | Broadcasts incoming data to all connected devices. |
| **Bridge** | Connects network segments and filters traffic using MAC addresses. |
| **Switch** | Intelligently forwards frames using MAC addresses. |
| **Gateway** | Connects different networks and performs protocol translation. |
| **Router** | Routes packets between different networks using IP addresses. |
| **Modem** | Converts digital and analog signals to enable Internet connectivity. |

# 📌 OSI Layer Mapping

| Device | OSI Layer |
|----------|-----------|
| NIC | Layer 1 & Layer 2 |
| Repeater | Layer 1 |
| Hub | Layer 1 |
| Bridge | Layer 2 |
| Switch | Layer 2 |
| Gateway | Multiple Layers (depends on implementation) |
| Router | Layer 3 |
| Modem | Layer 1 |

---

# 🛡 Security Perspective

Understanding networking devices is essential for cybersecurity because many attacks target network infrastructure.

Examples include:

- MAC Flooding
- ARP Spoofing
- Rogue Gateways
- Router Exploitation
- DHCP Attacks
- VLAN Attacks
- Man-in-the-Middle (MITM)

A solid understanding of networking devices helps security professionals identify, analyze, and mitigate these threats.
