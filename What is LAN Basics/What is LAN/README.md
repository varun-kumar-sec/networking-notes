# What is LAN (Local Area Network)
# 📖 Overview

A **Local Area Network (LAN)** is a computer network that connects multiple devices within a **limited geographical area**, such as a home, office, school, laboratory, or campus.

LAN enables connected devices to communicate with each other and share resources such as files, printers, applications, and Internet access at high speed.

Compared to other network types, LAN provides faster communication, lower latency, and easier management.

# 🌐 What is LAN?

**LAN (Local Area Network)** is a network that connects computers and other networking devices within a **small geographical area**.

Devices connected to a LAN can exchange data quickly and efficiently using wired or wireless communication.

A LAN is usually owned, managed, and maintained by a single individual or organization.

Examples include:

- Home Network
- Office Network
- School Computer Lab
- College Campus
- Small Business Network

---

# ❓ Why is LAN Used?

A LAN is used to enable communication and resource sharing between connected devices.

It allows users to:

- Share files
- Share printers
- Share Internet connections
- Access shared applications
- Communicate efficiently within the network

Because the communication occurs over a small area, LAN offers high performance and reliable connectivity.

---

# 📌 Characteristics of LAN

A Local Area Network has the following characteristics:

- Covers a limited geographical area
- Provides high-speed communication
- Low latency
- Privately owned and managed
- Easy to install and maintain
- Supports resource sharing
- Reliable and secure communication

# ✅ Advantages

- High data transfer speed
- Easy resource sharing
- Cost-effective
- Reliable communication
- Easy network management
- Supports Internet sharing

---

# ❌ Disadvantages

- Limited geographical coverage
- Initial setup cost
- Network failure may affect all connected devices
- Requires administration and maintenance
