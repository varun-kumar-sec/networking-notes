# What is LAN Basics
# 📖 Overview

A **Local Area Network (LAN)** is a network that connects multiple devices within a limited geographical area, such as a home, office, school, or campus.

LANs enable devices to communicate efficiently, share resources, and access network services through wired or wireless communication.

This module introduces the fundamental concepts of LANs, including their transmission media, connectors, and the basic process of setting up a Local Area Network.

Understanding LAN fundamentals is an essential step before learning advanced networking concepts such as Ethernet, Switching, Routing, IP Addressing, and the OSI Model.
