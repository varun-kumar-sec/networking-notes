# LAN Transmission Media
# 📖 Overview

**LAN Transmission Media** refers to the physical medium through which data travels between devices in a **Local Area Network (LAN)**.

Whenever two or more devices communicate within a LAN, data must travel through a communication medium. In wired LANs, this medium consists of physical cables that provide a reliable path for data transmission.

The choice of transmission media affects the network's:

- Speed
- Bandwidth
- Transmission Distance
- Reliability
- Cost
- Overall Performance

# 🌐 What is LAN Transmission Media?

LAN Transmission Media is the **physical communication channel** that carries data between devices connected in a Local Area Network.

It provides a dedicated path for transmitting data packets from one device to another, ensuring reliable and efficient communication.

Without transmission media, devices in a wired LAN cannot exchange information.

---

# ❓ Why is Transmission Media Required?

Transmission media is essential because it:

- Connects networking devices together
- Carries data from one device to another
- Enables communication within the LAN
- Supports resource sharing
- Provides stable and reliable connectivity

The type of transmission media selected depends on the network's performance and budget requirements.

---

# 📂 Types of Transmission Media Used in LAN

## 1. Coaxial Cable

Coaxial cables were widely used in early LANs and are still used in specific communication systems.

Characteristics:

- Electrical signal transmission
- Good resistance to interference
- Moderate transmission speed
- Longer distance than ordinary copper cables

## 2. Twisted Pair Cable

Twisted Pair Cable is the **most commonly used transmission medium in modern LANs**.

Characteristics:

- Uses copper wires
- Supports Ethernet communication
- Low cost
- Easy to install
- Available as UTP and STP

📖 Detailed Documentation:

```

└── Guided Transmission Media/
    └── Twisted Pair Cable
```

## 3. Fiber Optic Cable

Fiber Optic Cable transmits data using **light signals** instead of electrical signals.

Characteristics:

- Extremely high speed
- Very high bandwidth
- Long-distance communication
- Immune to electromagnetic interference (EMI)

| Distance | Short | Medium | Long |
| EMI Resistance | Moderate | High | Excellent |

# 💡 Choosing the Right Transmission Medium

The choice of transmission media depends on several factors:

- Required network speed
- Communication distance
- Installation cost
- Network environment
- Budget
- Future scalability

For example:

- **Twisted Pair Cable** is ideal for home and office LANs.
- **Fiber Optic Cable** is preferred for high-speed enterprise and data center networks.
- **Coaxial Cable** is generally used in specialized communication systems and legacy networks.
