# Presentation Layer
# 📖 Overview

The **Presentation Layer** is the **6th layer** of the OSI Model. It is responsible for **data translation, encryption, decryption, compression, and decompression**.

It acts as a **translator** between the Application Layer and the lower layers of the OSI Model by ensuring that data sent from one system can be correctly interpreted by another.

This layer is also responsible for securing data using encryption before transmission and decrypting it after reception.

# 🌐 What is the Presentation Layer?

The **Presentation Layer** is responsible for preparing data so that it can be understood by the receiving system.

Since different computers may use different data formats or character encodings, this layer converts data into a common format before transmission.

It also encrypts sensitive information before sending it and decrypts it after receiving it.

---

# ⚙️ Functions of the Presentation Layer

The Presentation Layer performs several important functions:

- Data Translation
- Data Encryption
- Data Decryption
- Data Compression
- Data Decompression
- Character Encoding Conversion
- Data Formatting

---

# 🔄 Common Formats & Standards

The Presentation Layer works with various data formats and encoding standards.

| Format / Standard | Purpose |
|-------------------|----------|
| ASCII | Character encoding |
| Unicode (UTF-8) | Universal character encoding |
| JPEG | Image compression |
| PNG | Image format |
| GIF | Image format |
| MPEG | Video compression |
| MP3 | Audio compression |
| SSL/TLS | Encryption for secure communication |

- The Presentation Layer is **Layer 6** of the OSI Model.
