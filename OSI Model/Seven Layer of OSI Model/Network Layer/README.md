# Network Layer
# 📖 Overview

The **Network Layer** is the **3rd layer** of the OSI Model. It is responsible for **logical addressing**, **routing**, and **path determination**, allowing data to travel between different networks.

Unlike the Data Link Layer, which delivers data only within the same local network, the Network Layer enables communication between devices located on different networks using **IP addresses**.

It determines the best path for data to travel from the source to the destination.

# 🌐 What is the Network Layer?

The **Network Layer** is responsible for delivering data **from the source network to the destination network**.

It uses **logical addresses (IP addresses)** to uniquely identify devices and determines the most efficient route for forwarding data across interconnected networks.

At this layer, data is organized into **packets** before being transmitted.

---

# ⚙️ Functions of the Network Layer

The Network Layer performs several important functions:

- Assigns and uses **IP addresses** for device identification.
- Determines the best path for data transmission (Routing).
- Forwards packets between different networks.
- Performs logical addressing.
- Supports internetwork communication.
- Handles packet fragmentation and reassembly (when required).

---

# 🖥️ Devices Operating at the Network Layer

Common networking devices operating at Layer 3 include:

- Router
- Layer 3 Switch (Multilayer Switch)

These devices forward data using **IP addresses**.

---

# 📡 Common Protocols

Some commonly used Network Layer protocols include:

| Protocol | Purpose |
|----------|----------|
| IPv4 | Logical addressing for network communication |
| IPv6 | Next-generation Internet Protocol |
| ICMP | Error reporting and network diagnostics |
| IGMP | Multicast group management |
| IPsec | Secure IP communication |

# 📌 Key Takeaways

- The Network Layer is **Layer 3** of the OSI Model.
- It is responsible for **logical addressing** and **routing**.
- Data at this layer is called a **packet**.
- It uses **IP addresses** to identify devices across different networks.
- Routers and Layer 3 switches operate at this layer.
- It enables communication between different networks.
