# OSI Model
# 📖 Overview

The **OSI (Open Systems Interconnection) Model** is a **seven-layer conceptual framework** developed by the **International Organization for Standardization (ISO)** to standardize network communication.

Instead of treating communication as one complex process, the OSI Model divides it into **seven logical layers**, where each layer performs a specific networking function.

Although modern networks primarily use the **TCP/IP Model**, the OSI Model remains one of the most important concepts for learning computer networking because it simplifies network design, protocol understanding, and troubleshooting.

# 📚 Topics Covered

| Topic | Description |
|--------|-------------|
| What is OSI Model | Introduction, purpose, importance, and features of the OSI Model |
| Seven Layers of OSI Model | Overview of all seven layers |
| Physical Layer | Transmission of raw bits over the communication medium |
| Data Link Layer | Framing, MAC addressing, and error detection |
| Network Layer | Routing and logical addressing using IP addresses |
| Transport Layer | End-to-end communication, segmentation, TCP/UDP |
| Session Layer | Session establishment, maintenance, and termination |
| Presentation Layer | Translation, encryption, and compression |
| Application Layer | Network services for end-user applications |
| Data Encapsulation | Data flow through the OSI layers during transmission and reception |

---

# 🏗️ The Seven Layers

The OSI Model consists of the following seven layers:

| Layer | Name |
|------:|------|
| 7 | Application Layer |
| 6 | Presentation Layer |
| 5 | Session Layer |
| 4 | Transport Layer |
| 3 | Network Layer |
| 2 | Data Link Layer |
| 1 | Physical Layer |

Each layer communicates with the layer directly above and below it, allowing network communication to be modular and easier to manage.

# 🌍 Why Learn the OSI Model?

Understanding the OSI Model helps you:

- Learn networking in a structured manner.
- Understand how data travels across a network.
- Troubleshoot network problems layer by layer.
- Understand the purpose of networking protocols.
- Prepare for networking certifications such as **CCNA** and **CompTIA Network+**.

# 🚀 Next Module

➡️ **TCP/IP Model**

After understanding the OSI Model, the next step is learning the **TCP/IP Model**, which is the practical networking model used on the Internet today.

The TCP/IP Model explains how modern devices communicate over networks using protocols such as **TCP**, **UDP**, **IP**, **HTTP**, and many others. It also introduces the comparison between the OSI Model and the TCP/IP Model.
