# 🌐 Internet Layer Protocols

The **Internet Layer** is responsible for enabling communication between devices across one or more interconnected networks. It provides **logical addressing**, **packet routing**, and **address resolution**, allowing data to travel from a source device to its destination.

Protocols operating at this layer ensure that data packets are properly identified, forwarded, and delivered to the correct network destination.

This directory covers the two fundamental Internet Layer protocols:

- **ARP (Address Resolution Protocol)** – Resolves an IPv4 address into a MAC address within a Local Area Network (LAN).
- **IP (Internet Protocol)** – Provides logical addressing and routes packets between devices and networks.

Together, these protocols form the foundation of communication in IPv4-based networks.
